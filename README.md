# Brandlight Payroll → חשבשבת

אפליקציית Streamlit להמרת דוחות שכר US Payroll לפקודות יומן בפורמט חשבשבת.

## התקנה

```bash
pip install -r requirements.txt
```

## הפעלה

```bash
streamlit run app.py
```

## מה האפליקציה עושה

1. **קליטת PDF** — קבצי US Payroll Consolidated (אחד או מרובים)
2. **חילוץ אוטומטי** — חשבוניות, רכיבים, עובדים, סכומים
3. **אימות** — סכום רכיבים = סה"כ חשבונית
4. **פקודת יומן** — גיליון בפורמט חשבשבת (טווח שטוח, SALARIES)
5. **היסטוריה** — שמירת חשבוניות + עובדים, סיכום YTD מצטבר
6. **דוח מצטבר** — 5 גיליונות: תקופה, עובד לפי תקופה, עובד מצטבר, התפלגות רכיבים, התפלגות חודשית

## מבנה פקודת יומן

| תאריך | חשבון חובה 1 | חשבון זכות 1 | חשבון זכות 2 | פרטים | אסמכתא | חובה מט"ח | זכות מט"ח |
|---|---|---|---|---|---|---|---|

- טווח: SALARIES
- תאריך: DD/MM/YYYY
- חשבונות, אסמכתא: טקסט
- שורות עם סכום 0 לא נכללות
- ללא שורות סיכום (טווח נתונים שטוח)
