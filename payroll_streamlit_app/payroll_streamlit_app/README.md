# Brandlight Payroll → חשבשבת

אפליקציית Streamlit להמרת דוחות שכר US Payroll לפקודות יומן בפורמט חשבשבת.

## התקנה

```bash
pip install -r requirements.txt
```

## הפעלה

```bash
streamlit run app.py
```

## מה האפליקציה עושה

1. **קליטת PDF** — מקבלת קבצי US Payroll Consolidated PDF (מרובים בו-זמנית)
2. **חילוץ אוטומטי** — מזהה חשבוניות, רכיבים, סכומים, תאריכים
3. **אימות** — בודקת שסכום הרכיבים = סה"כ חשבונית
4. **פקודת יומן** — יוצרת גיליון בפורמט חשבשבת (טווח נתונים שטוח, בלי סיכומים)
5. **היסטוריה** — שומרת חשבוניות קודמות ומציגה סיכום YTD מצטבר
6. **הורדת Excel** — 3 גיליונות: פקודת יומן + סיכום תקופה + מצטבר YTD

## מבנה פקודת יומן

| תאריך | חשבון חובה 1 | חשבון זכות 1 | חשבון זכות 2 | פרטים | אסמכתא | חובה מט"ח | זכות מט"ח |
|---|---|---|---|---|---|---|---|

- טווח: SALARIES
- תאריך: DD/MM/YYYY
- חשבונות, אסמכתא: טקסט
- שורות עם סכום 0 לא נכללות
- ללא שורות סיכום (טווח נתונים שטוח)
